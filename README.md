# Casa Serralarga Bookings System

## 🏨 Overview

Een compleet boeking- en beheerssysteem voor Casa Serralarga B&B met 6 kamers.

### Functies:
✅ Gast boeking interface (kalender + beschikbaarheid)
✅ Admin dashboard (reserveringen beheren)
✅ Dynamische prijzen per datum/periode
✅ Email bevestigingen
✅ Check-in/check-out systeem
✅ Factuur generatie
✅ Elegante Portugese B&B styling

## 🚀 Installatie

### Vereisten:
- Node.js 14+
- npm

### Stappen:
1. Clone de repository
2. Installeer dependencies:
   ```bash
   npm install
   ```
3. Configureer email settings in `server.js`
4. Start de server:
   ```bash
   npm start
   ```
5. Open http://localhost:3000

## 📁 Projectstructuur

```
casaserralarga-bookings/
├── index.html          # Gast boeking pagina
├── admin.html          # Admin dashboard
```
